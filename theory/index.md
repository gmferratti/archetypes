# theory/ — Teoria Avançada

Este diretório contém o arcabouço teórico que fundamenta as análises. Os arquivos são referencias de consulta durante a análise — não treinamento de arquétipos individuais, mas teoria que informa a *metodologia* analítica.

## Estrutura

```
theory/
├── index.md                    # Este arquivo — mapa do território
├── jung.md                     # Psicologia analítica — conceitos centrais
├── frye.md                     # Crítica arquetípica — Anatomy of Criticism
├── campbell.md                 # A Jornada do Herói — monomito
├── neumann.md                  # Grande Mãe e psicologia do feminino
└── methodology.md              # Como aplicar a teoria na análise literária
```

## Hierarquia de autoridade teórica

Para este projeto, a hierarquia de referência teórica é:

1. **Jung** — autoridade primária para os arquétipos psicológicos (Anima, Animus, Sombra, Si-mesmo, Persona)
2. **Campbell** — autoridade primária para a estrutura narrativa heroica (monomito)
3. **Frye** — autoridade primária para os padrões literários (modos, mitos, ciclos)
4. **Neumann** — autoridade complementar para o feminino arquetípico (Grande Mãe)
5. **Hillman** — psicologia arquetipal pós-junguiana (perspectivas alternativas)

## Princípio de uso

A teoria deve ser usada como **lente analítica**, não como **checklist de confirmação**. O risco de "ver forma em nuvem" (pareidolia analítica) aumenta quando o analista começa pelo arquétipo que quer encontrar e busca justificativas no texto — em vez de começar pelo texto e deixar que os padrões emirjam.

**Procedimento correto**:
1. Ler o texto sem hipótese arquetípica prévia
2. Identificar padrões de intensidade simbólica, estrutura narrativa, recorrência
3. Consultar definições e verificar se os critérios de evidência são satisfeitos
4. Nomear o arquétipo com grau de confiança
5. Citar os excertos que sustentam a conclusão
6. Documentar o que foi descartado e por quê
