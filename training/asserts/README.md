# training/asserts/ — Exemplos Literários Anotados

Este diretório contém exemplos de obras literárias que ilustram cada arquétipo. Cada arquivo segue a estrutura abaixo para garantir rastreabilidade e rigor analítico.

## Estrutura de cada arquivo de exemplo

```
training/asserts/<arquetipo>/<obra-autor>.md
```

## Template de anotação

```markdown
---
archetype: [nome do arquétipo]
work: [título da obra]
author: [autor]
year: [ano]
tradition: [tradição literária / país]
confidence: high | medium | low
---

## Contexto da obra

[Breve contextualização: gênero, período, tradição literária]

## Manifestação do arquétipo

[Descrição de como o arquétipo se manifesta na obra]

## Excertos de evidência

### Excerto 1
> [citação exata]
— [localização: capítulo, parte, página de edição de referência]

**Análise**: [por que este excerto evidencia o arquétipo — qual característica
identificadora está presente]

### Excerto 2
[...]

## Reasoning completo

[Cadeia argumentativa: das evidências à conclusão arquetípica. Deve incluir:
- Qual definição teórica está sendo aplicada
- Por que os excertos satisfazem os critérios de alta/média confiança
- O que foi considerado e descartado]

## Contra-indicações avaliadas

[O que poderia indicar outro arquétipo e por que foi descartado]
```
